## 🚀 Projects I've Built

### 🛒 Cart Super Add-On (CSAO) Recommendation System
A food add-on recommendation engine that suggests items to customers based on their behavior, item popularity, and restaurant context. Built using **LightGBM** as a binary classification model, with negative sampling and class balancing to handle imbalanced data. Evaluation was done using AUC, Precision@K, and NDCG@K. Deployed as a **Streamlit dashboard** giving personalized veg/non-veg recommendations per customer and restaurant.

---

### 🔍 Crime Intelligence System
A crime trend forecasting system using **Random Forest and XGBoost** regression. Handled full preprocessing, feature engineering, and hyperparameter tuning. Reached **92.83% model accuracy**. Built an interactive **Streamlit dashboard** for visualizing real-time crime trend insights.

---

### 🪪 Aadhaar Enrolment Intelligence Analytics
Analyzed over **9.8 lakh UIDAI enrolment records** across all Indian states and 700+ districts. Uncovered age-wise trends, regional gaps, and monthly patterns through statistical analysis. Evaluated biometric and demographic update load ratios to flag operational inefficiencies. Delivered **policy-ready dashboards** for smarter enrolment planning.

---

### 💼 Career Skill Gap Intelligence System
A platform that compares a person's skill set against industry benchmarks for specific job roles. Highlights the gaps between where someone is and where the market expects them to be — combining skills, projects, and career goals into one unified analytical view.
